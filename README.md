## 🚀 How to Run Locally

1. Clone the repository

```
git clone https://github.com/Rahul70079/Rapido-Clone---Bike-Taxi-App-UI
```

2. Open the folder

```
cd rapido-clone
```

3. Run the project

Open `index.html` in your browser.
